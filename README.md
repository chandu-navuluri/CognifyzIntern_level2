
# 🍽️ Restaurant Data Analysis - Level 2

## 📌 Project Overview

This project was completed as part of the Data Science Internship at Cognifyz Technologies.

The objective was to perform data analysis and feature engineering on a restaurant dataset to extract meaningful insights related to table booking, online delivery services, pricing patterns, and restaurant characteristics.

---

## 🎯 Tasks Completed

### Task 1: Table Booking and Online Delivery Analysis

- Calculated the percentage of restaurants offering:
  - Table Booking
  - Online Delivery
- Compared average ratings of restaurants with and without table booking.
- Analyzed online delivery availability across different price ranges.
- Created visualizations to support findings.

### Task 2: Price Range Analysis

- Identified the most common price range among restaurants.
- Calculated average ratings for each price range.
- Determined which rating color corresponds to the highest average rating.
- Visualized price range distributions and ratings.

### Task 3: Feature Engineering

- Created new features:
  - Restaurant Name Length
  - Address Length
- Encoded categorical variables:
  - Has Table Booking
  - Has Online Delivery
- Generated a machine-learning-ready dataset.
